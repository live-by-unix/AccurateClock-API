AccurateClock API

The ultra‑minimal, ultra‑precise public time API.

Overview

AccurateClock API provides globally‑distributed, low‑latency, NTP‑backed timestamps from Cloudflare’s edge network. No keys, no auth, no rate limits — just accurate time.

Endpoints

/api/time

GET https://accurateclock-api-public.live-by-unix.workers.dev/api/time
{
  "epoch_ms": 1710000000000,
  "iso_utc": "2026-06-24T18:00:00.000Z",
  "latency_ms": 2.8,
  "region": "SFO",
  "version": "1"
}

How to Use

JavaScript

const r = await fetch(
  "https://accurateclock-api-public.live-by-unix.workers.dev/api/time"
);
const d = await r.json();
console.log(d.epoch_ms);

Python

import requests

d = requests.get(
    "https://accurateclock-api-public.live-by-unix.workers.dev/api/time"
).json()

print(d["epoch_ms"])

cURL

curl https://accurateclock-api-public.live-by-unix.workers.dev/api/time
